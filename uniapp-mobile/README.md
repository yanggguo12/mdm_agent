## 🛠️ 技术栈与架构设计

- **框架内核**: Vue 3.x (Composition API `<script setup>` 响应式引擎)
- **多端跨端**: uni-app 标准规范 (一套代码完美编译为 **微信小程序**、支付宝小程序、iOS/Android App、H5 网页)
- **标签对齐**: 使用跨端组件标准标签 (例如 `<view>` 替代 `<div>`，`<text>` 替代 `<span>`, `<scroll-view>` 替代浏览器滚动条，从而保障小程序环境下不闪退，流畅排版)
- **视觉风格**: **极致高仿真微信原生组件设计**（包含模拟微信右上角胶囊按键操作、微信绿色健康主题、一键极速修复交互、本地体检历史对账等，完美符合 44px 舒适触控指标）
- **性能优化**: 所有组件图标采用**高解析度原生内嵌矢量 SVG**，体积小于 2KB，免去跨域加载 icon 字体产生乱码或慢加载的痛点。

## 🚀 极速部署使用指南

### 方法 A: 使用 HBuilderX 编译器导入运行 (推荐)
1. 下载或解压 `/uniapp-mobile` 目录。
2. 打开 **HBuilderX** -> 选择菜单栏 **「文件」 -> 「导入」 -> 「从本地目录导入」**。
3. 导入成功后，在菜单栏选择 **「运行」 -> 「运行到小程序模拟器」 -> 「微信开发者工具」**。
4. 运行开始后，HBuilderX 会自动编译并在后台唤醒微信官方开发者工具，您就可以在小程序真机模拟器中看到完全运行的小程序！

### 方法 B: 使用 命令行 & VS Code 运行
1. 打开终端进入该目录：
   ```bash
   cd uniapp-mobile
   ```
2. 安装依赖：
   ```bash
   npm install
   ```
3. 启动本地微信小程序编译监视：
   ```bash
   npm run dev:mp-weixin
   ```
4. 打开 **微信开发者工具** -> 选择 **「导入项目」** -> 选择生成的 `dist/dev/mp-weixin` 临时分发目录即能打开预览和真机调试！

## ✨ 微信端特色功能自愈细节说明

1. **微信多物理分区诊断**: 支持动态单选或复选 MARA 基础、MARC 工厂、MBEW 财务和 BOM 物理结构等，过滤算子精准到位。
2. **三段式雷达扫描对仗**: 还原真实智能体检雷达旋转进度，并实时向本地体检历史簿写入当前打分。
3. **AI数据特工高真会话**: 集成微信风格的聊天会话面板，带滚动穿透屏蔽以及自适应溢出文字折叠控制。
4. **胶囊刷新控制**: 提供了完美的微信胶囊模拟设计，点击右上角"同位刷按钮"能立刻弹出 Modal 询问是否复位或清空内存数据库状态。
